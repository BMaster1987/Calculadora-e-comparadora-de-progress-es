# Interactive Progressions Laboratory

## Project Overview
This project is an interactive offline educational web application focused on mathematical progressions for high school students.

It was developed as a teaching and learning resource to support the study of:
- Arithmetic Progression (A.P.)
- Geometric Progression (G.P.)
- Harmonic Progression (H.P.)

The application was designed to be visually engaging, didactic, and practical for classroom use and individual study.

## Educational Purpose
This project was created for educational purposes, especially for high school students who are learning sequence patterns, formulas, graphs, and comparisons between different types of progressions.

Its main goals are:
- to make abstract concepts easier to visualize
- to provide immediate numerical and graphical feedback
- to support exploration, comparison, and pattern recognition
- to serve as an interactive learning tool for mathematics classes

## What Was Implemented
### 1. Professional interface redesign
- Cleaner visual hierarchy
- More consistent layout
- Standardized buttons and action areas
- Improved readability and user navigation
- More polished educational app appearance

### 2. Separate laboratories for each progression
- Arithmetic Progression laboratory
- Geometric Progression laboratory
- Harmonic Progression laboratory

Each laboratory includes:
- input parameters
- formula display
- generated terms table
- graph visualization
- quick examples
- table search
- export options

### 3. Comparison module
- Added a selector to compare:
  - only two progressions
  - or all three progressions at the same time
- Improved comparative reading of the results
- Better support for identifying differences between progression behaviors

### 4. Sequence analyzer improvements
- Fixed parsing issues for Brazilian decimal notation
- Inputs now support decimal commas
- Better handling of manually typed sequences
- More reliable behavior for student input

### 5. Graph improvements
- Better visual clarity
- Better label positioning
- Adjustments to avoid overlap between axis labels and numeric values
- Improved readability in all progression graphs
- More didactic presentation for interpretation

### 6. Brazilian decimal format support
- Numeric input follows Brazilian decimal conventions
- Decimal comma support was prioritized for usability in Brazil
- Input behavior was adapted for local educational context

### 7. Harmonic progression support
- Harmonic progression was added with features equivalent to the other progression modules
- Includes formulas, graph, generated table, examples, and exports
- Added educational interpretation through the associated arithmetic progression denominator idea

### 8. Quiz preservation
- The quiz section was intentionally kept focused only on:
  - Arithmetic Progression
  - Geometric Progression
- No major structural changes were made there, as requested

## Technical Notes

- The interface and logic were refined across multiple iterations
